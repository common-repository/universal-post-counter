== Description ==

= UNIVERSAL POST COUNTER =

Universal Post Counter by PressTigers is a simple plugin that allows admin to see quick stats of post count from all post types within the admin dashboard in the form of a dashboard widget along other dashboard widgets.

Below are some salient features:

== Universal Post Counter Features ==

*   Simple to use
*   100% responsive

As we continue to strive for perfection further features will be added in future updates.

== Installation ==

1.  Download plugin.
2.  Upload 'universal-post-counter.zip' to the '/wp-content/plugins/' directory to your web server.
3.  Activate the plugin through the 'Plugins' menu in WordPress.
4.  Visit WordPress Dashboard Area for plugin widget.

== Frequently Asked Questions ==

1. Will admin be able to see post count from both default and custom post types?
Yes, Universal Post Counter admin widget will show post count from all post types that has public paramater 'true'
